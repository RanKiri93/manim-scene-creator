    Multiline calculation
    
            
    
              Export uses Manim MathTex (LTR). Block: one MathTex (one string or
              multiple string args). Segmented (narration): split with ||| — each part is a
              separate MathTex argument (same horizontal line); ||| is only for animation and
              voice bookmarks, not a line break. Preview uses KaTeX; multline is mapped to
              aligned in preview only.
            
    
            
    
              
              
            
    
            
            
    
              
    
                
                
                
    
                  Multiple strings: each non-empty line becomes r"…". One string:
                  paste the full TeX; use multline wrap below if needed.
                
    
              
    
            
    
            
    
              
    
                Separate with ||| only to define narration / animation segments — export is one
                MathTex(r"…", r"…", …) on a single line (add \quad inside a segment if you want
                space). Voiceover: use ||| in the voice script in the same order.
